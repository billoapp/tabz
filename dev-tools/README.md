# Development Tools

This folder contains development and debugging tools that are not part of the production codebase.

## Structure

- `scripts/` - Development and debugging scripts
- `tests/` - Test files and test utilities
- `docs/` - Development documentation and notes
- `sql/` - Database scripts and migrations for development
- `debug/` - Debugging tools and temporary files

## Usage

These tools are for development purposes only and are gitignored to keep the main repository clean.
